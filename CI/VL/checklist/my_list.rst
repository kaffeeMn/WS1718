t-norm und andere normen
^^^^^^^^^^^^^^^^^^^^^^^^

t-norm is the intersection

* A1 t(a,1) = a
* A2 b <= c: t(a,b) <= t(a,c)
* A3 t(a,b) = t(b,a)
* A4 t(a, t(b,c)) = t(t(a,b),c)

s norm is the union

* A1 s(a,0) = a
* A2 b <= c: s(a,b) <= s(a,c)
* A3 s(a,b) = s(b,a)
* A4 s(a,s(b,c)) = s(s(a,b),c)

c norm is the compliment

* A1 c(0) = 1; c(1) = 0
* A2 a <= b: c(a) >= c(b)
